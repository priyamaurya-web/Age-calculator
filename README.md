# 🎂 Age Calculator

A beautiful, fully functional age calculator with live ticker, zodiac signs, life milestones, planet ages, numerology, and fun facts.

Built with pure HTML, CSS (Tailwind), and JavaScript — no backend required.

![Age Calculator Preview](https://age-calculator-hewz.onrender.com/)

## ✨ Features

- Real-time live age counter (years, days, hours, seconds)
- Western & Chinese Zodiac
- Life Path Number (Numerology)
- Generation badge
- Age on other planets
- Life progress meter (based on 72-year average)
- Next birthday countdown + confetti
- Fun life statistics
- Dark/Light mode
- Fully responsive & animated

## 🌐 Deployment

### Deploy on Render (Recommended - Free)

1. Push this repo to GitHub
2. Go to [Render.com](https://render.com) → New Static Site
3. Connect your GitHub repository
4. Use these settings:

   - **Build Command**: `echo 'No build required'`
   - **Publish Directory**: `.` (just a dot)

5. Click Deploy.
